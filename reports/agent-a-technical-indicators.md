## 三、最有效的技術指標黃金組合

### 【第一名】均線多頭排列 + MACD 黃金交叉

**組成指標：**
- 均線：MA5 > MA10 > MA20（多頭排列）
- MACD：MACD線由下往上穿越信號線

**進場閾值條件：**
- MA5 突破 MA10 且 MA10 > MA20（均線多頭排列確認）
- MACD DIF 線從負值區翻正並超越 MACD Signal 線
- 進場時機：當日收盤價高於前一日收盤價 0.5% 以上

**進場時機細則：**
- 最佳進場點：股價回測 MA5 或 MA10 支撐時進場
- 替代進場點：突破時立即追價（但需嚴設停損）

**出廠/停利閾值：**
- 5日短線：RSI > 75 或 獲利達 5-8% 考慮分批出廠
- 10日波段：MA5 死叉 MA10 或 MACD 柱狀體由正轉負
- 強制停損：跌破 MA20 或 虧損超過 7%

**篩選條件（做多勝率最高）：**
- 大盤指數同步多頭（加權指數 MA5 > MA10）
- 成交量較昨日放大 1.5 倍以上
- 股價位於布林通道中軌以上
- 排除 RSI > 80 的超買區（易回調）

### 【第二名】KD 低檔黃金交叉 + RSI 支撐確認

**組成指標：**
- KD：K值由下往上穿越 D值，且兩者均低於 30（水位低檔）
- RSI：RSI(14) 介於 35-55 之間（支撐確認）

**進場閾值條件：**
- K值 < 30 且 D值 < 30（低檔黃金交叉）
- K值由下往上穿越 D值（黃金交叉確認）
- RSI(14) > 40 且股價站上 5 日均線
- 成交量較前一日量增 20% 以上

**進場時機細則：**
- 當日 K 值收盤站上 D 值
- 個股 5 日均線向上的斜率 > 0（均線上揚）
- 最佳時機：開盤一小時內股價站穩均線之上

**出廠/停利閾值：**
- K值 > 80 且出現死亡交叉（高檔反轉）
- RSI(14) > 75 考慮分批離廠
- 5日操作：獲利 4-6% 積極了結
- 10日操作：MA10 跌破 MA20 才全部出清
- 強制停損：KD 死亡交叉當日收盤價低於進場價 3%

**篩選條件（做多勝率最高）：**
- 大盤未出現連續下跌走勢
- 成交量逐步溫和放大（量價配合）
- 選擇股性活潑、股本適中的個股（50-300 億）
- 避免在指數破底時進場

---

### 【第三名】成交量突破 + 布林通道下軌支撐

**組成指標：**
- 成交量：今日成交量為 5 日均量的 1.5 倍以上
- 布林通道：股價觸及或接近布林下軌（支撐確認）

**進場閾值條件：**
- 股價接進布林下軌（股價 ≤ BOLL下限）
- 當日成交量 > 5 日均量 1.5 倍（量能突破）
- 5 日均線開始走平或微微上揚
- 布林帶寬度（BandWidth）處於相對低點（壓縮整理接近尾聲）

**進場時機細則：**
- 當日股價從布林下軌反彈且收紅 K 棒
- 成交量在股價反彈時同步放大
- 確認支撐：隔日股價持續站穩布林中軌（MA20）

**出廠/停利閾值：**
- 股價碰觸布林上軌（偏離超過 2% 可考慮減碼）
- 5日操作：獲利達 6% 以上分批離場
- 10日操作：股價跌破 10 日均線且量能萎縮
- 強制停損：股價連續 2 日收盤低於布林下軌（破支撐）

**篩選條件（做多勝率最高）：**
- 前期有明顯量能萎縮（主力洗盤結束）
- 布林帶寬度由寬變窄（波動率壓縮）
- 選擇支撐力道強的族群或藍籌股
- 大盤當日非大跌走勢（避免破底）

---

### 【第四名】MA5 / MA10 黃金交叉 + RSI 未超買

**組成指標：**
- 移動均線：MA5 黃金交叉 MA10（價穩均線之上）
- RSI：RSI(14) 處於合理區間（40-65）

**進場閾值條件：**
- MA5 由下往上穿越 MA10（黃金交叉）
- 交叉時 MA5 與 MA10 之間距離越小越佳（減少假突破）
- 股價高於 MA5 且 MA5 > MA10 > MA20
- RSI(14) 處於 40-60 区間（非超買，進場空間大）
- 成交量較 5 日均量增加 30% 以上

**進場時機細則：**
- 黃金交叉當日收盤價確認後進場
- 若當日成交量 > 20 日均量 2 倍，突破機率更高
- 激進進場：開盤跳空高開且回踩不破 MA5

**出廠/停利閾值：**
- MA5 死亡交叉 MA10（短線多空翻轉）
- RSI(14) 向上突破 70 之後又反轉跌破 70（短線超買離場）
- 5日操作：最大獲利 5-7% 或跌破 MA5 收盤
- 10日操作：MA10 跌破 MA20 全數離場
- 強制停損：MA5 死亡交叉當日收盤價低於 MA10 超過 1%

**篩選條件（做多勝率最高）：**
- 交叉角度越陡峭（MA5 上揚角度 > 30 度）信號越強
- 交叉前股價已在 MA10 附近整理 3-5 日（蓄勢時間充足）
- 選擇強勢股（股價漲幅優於大盤）
- 避免在 RSI > 70 時進場（短期獲利了結壓力大）

---

### 【第五名】DMI趨向指標 + SAR 確認進場

**組成指標：**
- DMI：+DI 向上突破 -DI（多頭趨勢確認）
- SAR：拋物線轉向點在股價下方（做多訊號）

**進場閾值條件：**
- +DI > -DI 且 ADX > 20（趨勢明確）
- ADX 持續上揚代表趨勢正在增強
- SAR 點位於股價下方（停損點不斷上移保護獲利）
- 進場時機：SAR 由上方翻至下方，代表多頭啟動

**進場時機細則：**
- 當日 SAR 指標由紅翻綠（股價在 SAR 之上）
- +DI 黃金交叉 -DI 且成交量配合放大
- 最佳買點：股價回踩 SAR 支撐時進場（提高風報比）

**出廠/停利閾值：**
- SAR 由綠翻紅（多單止損離場）
- -DI 向上突破 +DI（空頭訊號浮現）
- ADX 由高檔下滑 > 5 個單位（趨勢減弱）
- 5日操作：獲利達 4-6% 考慮獲利了結
- 10日操作：嚴格以 SAR 移動停損（每日益動上調）
- 強制停損：股價跌破 SAR 當日收盤價

**篩選條件（做多勝率最高）：**
- ADX > 25 且持續上升（趨勢強度足夠）
- +DI 與 -DI 之間的差距越大越好
- 選擇中期均線多頭排列的個股
- 避免在盤整區間（ADX < 15）使用此指標

## 四、具體篩選條件（偽代碼）

### 進場條件

```
FUNCTION 篩選做多標的(股票):
    // 必要條件
    IF NOT (加權指數.MA5 > 加權指數.MA10 > 加權指數.MA20):
        RETURN FALSE  // 大盤必須多頭排列
    
    IF NOT (股價 > MA5 AND MA5 > MA10 AND MA10 > MA20):
        RETURN FALSE  // 股價必須站上均線多頭排列
    
    // 動能條件
    IF NOT (KD.K值 > 30 AND KD.K值 > KD.D值):
        RETURN FALSE  // KD 需從低檔翻升
    
    IF NOT (RSI(14) >= 40 AND RSI(14) <= 65):
        RETURN FALSE  // RSI 不可超買或超賣
    
    IF NOT (成交量 > 5日均量 * 1.5):
        RETURN FALSE  // 成交量需明顯放大
    
    // 確認條件
    IF NOT (MACD.DIF > MACD.Signal AND MACD.柱狀體 > 0):
        RETURN FALSE  // MACD 需確認多頭
    
    IF NOT (股價處於布林通道中軌以上):
        RETURN FALSE
    
    RETURN TRUE  // 通過所有條件
```

### 持有期間條件

```
FUNCTION 持有條件檢查(股票, 進場價):
    目前獲利 = (目前股價 - 進場價) / 進場價 * 100
    
    // 續抱條件
    IF 目前獲利 < 5%:
        IF (MA5 > MA10 AND 股價 > MA5 AND MACD.柱狀體 > 0):
            RETURN TRUE  // 續抱
    
    // 減碼條件
    IF (RSI > 75 OR 目前獲利 > 8%):
        RETURN "減碼"
    
    // 停損條件
    IF (股價 < MA20 OR 目前獲利 < -7%):
        RETURN FALSE  // 強制出廠
    
    RETURN TRUE
```

### 出廠條件

```
FUNCTION 出廠判斷(股票, 進場價):
    目前獲利 = (目前股價 - 進場價) / 進場價 * 100
    
    // 停利條件
    IF 目前獲利 >= 8%:
        RETURN "分批停利"
    
    IF RSI > 80:
        RETURN "RSI超買離場"
    
    IF MA5 死亡交叉 MA10:
        RETURN "均線空頭訊號"
    
    IF MACD.柱狀體 由正轉負:
        RETURN "MACD空頭訊號"
    
    // 停損條件
    IF 目前獲利 < -7%:
        RETURN "強制停損"
    
    IF 股價 < MA20:
        RETURN "跌破均線停損"
    
    RETURN "續抱"
```

## 五、組合策略總結與勝率優化

### 核心原則：五大指標黃金組合

| 角色 | 指標 | 用途 |
|------|------|------|
| 主指標 | 均線多頭排列 | 確認趨勢方向 |
| 確認指標 | MACD 黃金交叉 / KD 低檔黃金交叉 | 確認動能轉強 |
| 濾網指標 | 成交量放大 | 確認信號真實性 |
| 輔助指標 | RSI 合理區間 | 避開超買超賣區 |
| 移動停損 | SAR / 布林通道 | 動態保護獲利 |

### 短線勝率最高的篩選條件清單

1. **大盤環境：** 加權指數 MA5 > MA10 > MA20（多頭排列）
2. **價格位階：** 股價站上所有均線（MA5 > MA10 > MA20）
3. **均線型態：** MA5 與 MA10 黃金交叉且角度陡峭
4. **動能指標：** MACD 柱狀體由負轉正且放量
5. **KD 位置：** K 值由低檔翻升（< 30 → > 50）
6. **RSI 區間：** RSI(14) 介於 40-65（非超買/非超賣）
7. **成交量：** 當日成交量 > 5 日均量 1.5 倍
8. **布林狀態：** 股價站上中軌（MA20）且未觸上軌
9. **趨勢強度：** ADX > 20 且持續上升

### 最佳進場時機

- **盤前篩選：** 開盤 30 分鐘內量增價漲且站上均線
- **盤中確認：** 回踩均線支撐不破時加碼
- **尾盤操作：** RSI 在 50-60 之間且當日量增價漲

### 出廠紀律

- **停利：** 5日短線 5-8%，10日波段 10-15%
- **停損：** 單筆虧損不超過總資金 2-3%
- **移動停損：** 獲利率超過 5% 後以 SAR 追蹤停利
- **強制出廠：** MA5 死叉 MA10 或 MACD 柱狀體由正轉負

## 七、風險提示

1. 本研究報告為技術分析理論探討，非投資建議
2. 技術指標存在假訊號風險，任何單一指標不應作為唯一進場依據
3. 短線交易需嚴格風險管理，建議單筆虧損控制在總資金 2% 以內

5. 建議結合基本面分析與市場情緒綜合判斷
6. 歷史勝率不代表未來績效，實際交易需有風險意識
